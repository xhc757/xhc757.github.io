使用 Github Pages 和 Hugo 搭建个人博客

Github Pages 在 github 新建一个仓库，仓库名设置为 username.github.io(username必须是github的用户名)，稍等几分钟即可使用 username.github.io 访问 使用Hugo生成博客 需要在username.github.io上传我们的网站内容。我们可以手动制作网页，但为了方便，我们可以使用静态内容生成器，比如Peliacan。 安装Hugo 在windows下面安装Hugo只需要下载单个可执行文件，然后将其位置添加到PATH环境变量。打开powershell窗口，输入hugo version回车，如果显示hugo说明已经安装好了。 ##快速上手hugo 输入 hugo new site xhc757.github.io 回车，会创建文件夹xhc757.github.io，打开文件夹会看到生成了很多文件和文件夹 进入这个目录，输入 hugo new posts\fist-blog.md 回车，这样就生成了一篇博客，我们可以打开这个md文件，创作自己的博客 我们还需设置一个主题，下载一个hugo theme，放到 thems文件下，然后打开hugo.toml, 指定主题 接下来输入 hugo，就会生成网页，输出在 public文件夹，这个文件夹不会自动清空，如果有需要，我们可以在每次生成前手动清空Public文件 然后输入 hugo server 回车，在浏览器打开 http://localhost:1313/ 就可以预览博客 详细配置hugo 可以看到生成的网站非常简陋，接下来我们详细配置hugo 推送到Github
